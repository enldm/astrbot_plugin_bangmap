# AstrBot 邦邦地图插件

一个用于查询全国邦邦群（BanG Dream! 群组）信息的 AstrBot 插件。

## 功能

- 查询全国各省份的邦邦群信息
- 支持省份全称、简称、单字别名查询
- 自动缓存数据，减少 API 请求
- 兼容带后缀和不带后缀的省份名称

## 安装

1. 将插件放置到 AstrBot 的 `plugins` 目录
2. 重启 AstrBot 或使用插件管理命令加载插件

## 使用方法

### 查询指定省份的邦邦群

```
/邦邦地图 [省份]
```

**参数说明：**
- `省份`：可选项，支持以下输入格式
  - 省份全称：如 `广东`、`四川`、`江苏`
  - 省份简称：如 `粤`、`川`、`蜀`、`苏`
  - 直辖市：如 `北京`、`沪`、`渝`
  - 特别行政区：如 `香港`、`港`、`澳门`

**示例：**

```
/邦邦地图 川
/邦邦地图 四川
/邦邦地图 粤
/邦邦地图 广东
/邦邦地图 北京
/邦邦地图 海外
```

### 不带参数查询

```
/邦邦地图
```

返回帮助信息。

## 支持的省份

插件支持全国所有省、自治区、直辖市、特别行政区及海外地区的查询：

- **直辖市**：北京、天津、上海、重庆
- **省份**：河北、山西、辽宁、吉林、黑龙江、江苏、浙江、安徽、福建、江西、山东、河南、湖北、湖南、广东、海南、四川、贵州、云南、陕西、甘肃、青海、台湾
- **自治区**：内蒙古、广西、西藏、宁夏、新疆
- **特别行政区**：香港、澳门
- **其他**：海外
